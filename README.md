# Luis Merino - Research Portfolio

A modern, interactive portfolio showcasing research in Human-Robot Interaction, Trust in AI, and Computer Vision demos.

## Features

### 🎯 Research Showcase
- **Publications Section**: Links to research papers on trust in human-robot interaction
- **Research-Based Demos**: Interactive demonstrations based on actual research findings
- **Personality Assessment**: Big Five personality test with AI trust predictions

### 🤖 Interactive Computer Vision Demos
- **Emotion Detection**: Real-time facial emotion recognition using ML5.js
- **Pose Estimation**: Live human pose detection and skeletal tracking
- **Object Tracking**: Real-time object detection with YOLO simulation
- **Augmented Reality**: Interactive AR experience with gesture controls

### 🧠 Trust in AI Demonstrations
- **Trust Assessment**: Interactive scenarios based on research findings
- **Personality & Trust**: Big Five personality test predicting AI trust levels
- **Research Insights**: Evidence-based explanations of trust factors

### 🎨 Modern Design
- **Responsive Layout**: Works on desktop, tablet, and mobile devices
- **Smooth Animations**: Intersection Observer API for scroll animations
- **Modern Typography**: Inter font family for clean, readable text
- **Interactive Elements**: Hover effects and smooth transitions

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Computer Vision**: ML5.js, P5.js, AR.js
- **Styling**: Custom CSS with modern design patterns
- **Hosting**: GitHub Pages

## Research Background

This portfolio is based on research in:
- Human-Robot Interaction (HRI)
- Trust in Artificial Intelligence
- Individual Differences in Technology Adoption
- Personality Psychology and AI Trust

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. Navigate through the different sections and try the interactive demos

## Demo Requirements

- **Camera Access**: Required for computer vision demos
- **Modern Browser**: Chrome, Firefox, Safari, or Edge
- **Mobile Device**: Recommended for AR demo
- **HTTPS**: Required for camera access in production
